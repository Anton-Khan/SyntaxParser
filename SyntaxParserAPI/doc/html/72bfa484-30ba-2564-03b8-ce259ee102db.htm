SyntaxParserAPI - пространство имён

Библиотека позволяет решать арифметические выражения в инфиксной нотации, заданные строкой.
Классы
  КлассОписание
Analyzer
Класс анализирующий список токенов на соответствие грамматике языка.
DijkstraStackMachine
Класс собирающай токены из инфиксной в обратную польскую нотацию.
Lexem
Содержит все основные лексемы и группы лексем языка.
Parser
Класс позволяющий разбить выражиние на токены.
PostfixPolishNotation
Класс представляет список токенов в обратной польской нотации.
SyntaxParser

Основной класс для работы с синтаксическим парсером.

Token
Класс содержит информацию о полученной лексеме и значении.
