SyntaxParserParse - метод

Решает поданное на вход выражение.

Пространство имён:  SyntaxParserAPI
Сборка:  SyntaxParserAPI (в SyntaxParserAPI.dll) Версия: 1.0.0
Синтаксис
C#
Копировать
public static void Parse(
	string? expression,
	bool showParsingModule = false,
	bool showAnalizingModule = false,
	bool showPpnBuilderModule = false,
	bool showPpnSolverModule = false

Параметры

expression
Тип: SystemString

Тип: SystemBoolean

showAnalizingModule (Optional)
Тип: SystemBoolean

Тип: SystemBoolean

showPpnSolverModule (Optional)
Тип: SystemBoolean
При установке в true [Показывает работу модуля решающего PostfixPolishNotation ].
Заметки

Функция оследовательно обрабатыет поданное на вход выражение.
1. Парсит входное выражение, разбирая его на лексемы.
2. Полученные лексемы анализируются по правилам определенной грамматики.
3. По завершению анализа строится обратная польская нотация.
4. Обратная польская нотация последовательно решается.
